AI Sales Telegram Bot

AI Sales Telegram Bot — интеллектуальный Telegram-бот для автоматизации общения с клиентами и обработки заказов. Бот использует дообученную модель OpenAI GPT-4o Mini, обученную на данных компании, что позволяет предоставлять клиентам точную информацию о товарах, ценах, характеристиках и условиях доставки.

Возможности

* Автоматическое консультирование клиентов 24/7
* Использование дообученной AI-модели OpenAI
* Расчет стоимости товаров в режиме реального времени
* Поддержка истории диалога
* Автоматическая обработка запросов на заказ
* Предоставление информации о продукции и компании
* Ответы исключительно на русском языке
* Интеграция с Telegram Bot API

Установка и запуск

1. Клонирование репозитория

git clone https://github.com/your-username/ai-sales-telegram-bot.git
cd ai-sales-telegram-bot

2. Создание виртуального окружения

python -m venv .venv

Windows:

.venv\Scripts\activate

Linux/macOS:

source .venv/bin/activate

3. Установка зависимостей

pip install -r requirements.txt

4. Настройка переменных окружения

Создайте файл `.env`:

OPENAI_API_KEY=your_openai_api_key
TELEGRAM_BOT_TOKEN=your_telegram_bot_token

5. Запуск бота


python bot.py

Принцип работы

1. Клиент отправляет сообщение в Telegram.
2. Сообщение сохраняется в историю диалога.
3. Запрос отправляется в дообученную модель GPT-4o Mini.
4. Модель анализирует запрос и предоставляет ответ на основе базы знаний компании.
5. Бот отправляет ответ пользователю.

Особенности AI-модели

Дообученная модель обучена на внутренних данных компании и умеет:

* Консультировать по продукции
* Рассчитывать стоимость заказов
* Предоставлять технические характеристики товаров
* Информировать об условиях доставки
* Помогать клиентам оформлять заказы

Безопасность

Не публикуйте в открытом доступе:

* OPENAI_API_KEY
* TELEGRAM_BOT_TOKEN
* Другие конфиденциальные данные

Добавьте файл .env в .gitignore.
